# mediapip-gusture-control
基于mediapip的手势操控完整实例

## 功能
* 食指伸直，中指无名指小拇指收拢，则鼠标跟随手移动
* 食指弯曲，则点击鼠标
* 手掌打开，然后指尖合拢，则用win+tab快捷键打开任务视图
